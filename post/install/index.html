安装正版office软件

2020-10-27

Contents

  • 前言
  • 前方高能警告
  • 后话

前言

office哪都好用，但缺点就是正版的office很贵（哦不，这不是office的缺点，是我的缺点）。但office在工作中又必不可少，给老板汇报、职级答辩写PPT，毕业论文写word，会计用execl做数据处理。 怎么办？装个盗版吧。

盗版在office 2007之前，都是可以用序列号激活，序列号在百度上一搜一大把。 但后面的office2010开始，微软用的是激活服务器的方式，office会定期连接微软服务器获取激活信息，让微软封号变得更容易了。 但难不倒中国人，各种奇奇怪怪得激活手段都有，比如有些就在本地安装一个程序，模拟微软的激活服务器。

但这些方式都太麻烦了，找一个可用的程序都要找半天，而且还有中病毒的风险。

前方高能警告

我这里推荐一个方式，在淘宝上直接搜索ofiice关键字，可搜到一些卖号的商家，挑一个最便宜的就行。卖号的分不同的版本，不同版本价格不一样。不过都非常便宜，一般几块钱就可以了，我的office 365就是7块钱买的。而且这是通过号激活的，没有中病毒的危险。安全可靠，7块钱，激活不了也不吃亏啊。

后话

虽然不知道号是从哪来的，但亲测可用，免去了折腾的麻烦。 这么强大的淘宝，还是有很多人都不知道的。除了office，其他正版软件都可以卖到，而且便宜得你不信啊。
